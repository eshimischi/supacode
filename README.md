# supacode

A native macOS app that orchestrates multiple coding agents simultaneously, enabling parallel execution of 50+ agents.

![Supacode](./assets/screenshot.png)

## Features

- **Native Performance** - Blazing fast macOS app powered by libghostty
- **BYOA (Bring Your Own Agents)** - Supports Claude Code, Codex, and Opencode
- **Git Worktree Integration** - Isolated workspaces per task
- **GitHub Integration** - PR management, CI checks, and conflict resolution
- **No Translation Layers** - Agents run directly in terminal
